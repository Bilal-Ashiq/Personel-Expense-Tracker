The Expense Tracker is a full-stack web application designed to help users manage their personal finances efficiently. It allows users to securely register and log in, then track their income and expenses in real-time. Users can create, edit, and delete transactions, categorize them for better organization, and view detailed financial summaries on a dynamic dashboard. Interactive charts provide visual insights into spending patterns, helping users make informed financial decisions. This project demonstrates full-stack MERN development, secure authentication, CRUD operations, and real-world data handling.
